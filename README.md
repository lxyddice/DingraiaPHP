# 这是什么？

这个是一个关于运行钉钉机器人的代码，用来运行Webhook的钉钉机器人功能，框架中有部分小工具。
只实现了20%的功能，将来可能会完善。
作者是PHP废物，请不要催，你行你上。

# 特点

使用简单，插件好写（php是这样的），支持At

# 注意

本项目适用于一般聊天机器人开发，不建议用于生产环境

At需要传入userid

作者自身已经挂着该框架，保证框架确实可用 ~~如果懒癌发作不更新的另当别论~~

**自带文档可能更新不及时！（懒）**

**本项目基于企业内部机器人开发，使用企业内部机器人以获得更好的体验**

# 需求

需要公网可访问的服务器，还得拥有企业内部开发权限
